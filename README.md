# StockFlow API

StockFlow é um sistema de controle de estoque desenvolvido para gerenciar entradas, saídas e monitoramento de produtos de forma simples, eficiente e organizada.

## Funcionalidades

- Cadastro de produtos
- Atualização de entrada e saída de estoque
- Consulta de produtos
- Controle de quantidade em tempo real
- Organização por categorias
- Histórico básico de movimentações

## Tecnologias

- Java 21
- Spring Boot
- Maven 
- PostgreSQL
- JPA
- Lombok
